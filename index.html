Stack-Overflow: Semantic Relationship Analysis

Master Period, Southwest University
November 2019 - October 2020
The data comes from the programming community Stack-Overflow which has many technical terms and questions. Each technical question has multi-technical terms to label it and each term has a definition text to introduce it. So there are many complex relations in these technical terms. In this case, my research is to recognize these specific semantic relations and try to construct a semantic graph of programming technology, which would help us to understand the evolution of programming technology.
We try to identify the semantic relation of two technical terms via their hypernyms. For example, the term pair "Python-Numpy" can be labeled as â€œMember-Collection" via hypernym pair "Language-Library". In this case, we proposed a novel hypernym extraction method based on the text classification model to extract the hypernym-hyponym relation of technical terms. Link
